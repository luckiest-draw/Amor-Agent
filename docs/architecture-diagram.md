## 二、Orchestrator + Runner 执行流

```
                    用户任务
                       │
                       ▼
              ┌─────────────────┐
              │  Orchestrator    │  单 Agent 模式（仿 Claude Code）
              │  execute(task)   │  没有预分类，模型自己决定
              └────────┬────────┘
                       │
                       ▼
              ┌─────────────────┐
              │     Runner       │
              │  Think→Act→      │
              │  Observe→Judge   │
              └────────┬────────┘
                       │
          ┌────────────┼────────────┐
          ▼            ▼            ▼
   ┌──────────┐ ┌──────────┐ ┌──────────┐
   │  简单任务  │ │ 搜索/工具  │ │ 复杂任务  │
   │  直接回答  │ │  单轮完成  │ │  调delegate│
   └──────────┘ └──────────┘ └─────┬────┘
                                   │
                    ┌──────────────┴──────────────┐
                    ▼                             ▼
             ┌──────────┐                  ┌──────────┐
             │ 研究员 Agent│                  │ 审核员 Agent│
             │ 搜索+整理   │                  │ 检查+退回   │
             └──────────┘                  └──────────┘
                    │                             │
                    └──────────┬──────────────────┘
                               ▼
                      ┌─────────────┐
                      │  汇总返回用户  │
                      └─────────────┘
```

## 三、Runner 核心循环

```
              ┌─────────────────┐
              │   messages[]    │  ← Assembler 装配
              │ (system+role    │
              │  +tools+file    │
              │  +rag+search    │
              │  +memory+history│
              │  +task)         │
              └────────┬────────┘
                       │
          ┌────────────▼────────────┐
          │  while 未完成            │
          │  and 步数 < max_steps    │
          └────────────┬────────────┘
                       │
          ┌────────────┼────────────┐
          ▼            ▼            ▼
   ┌────────────┐ ┌────────────┐ ┌────────────┐
   │ ① Think    │ │ ② Act      │ │ ③ Observe   │
   │ LLM 推理   │ │ 执行工具    │ │ 结果注入     │
   │ LiteLLM    │ │            │ │ messages    │
   │ tool_defs  │ │ ToolReg    │ │   .append   │
   │ 传给 LLM   │ │   .execute │ │             │
   │            │ │            │ │             │
   │ 返回       │ │ 中断审批?   │ │             │
   │ Thought   │ │ INTERRUPT  │ │             │
   └─────┬─────┘ └─────┬─────┘ └─────┬──────┘
         │              │              │
         │ tool_calls?  │              │
         ├── 无 ────────┴──────────────┴──→ ④ Judge: 任务完成 → 结束
         │                                        │
         └── 有 ──→ 循环继续 ──────────→ 回到 ①   │
                                                  │
                       ┌──────────────────────────┘
                       ▼
                ┌────────────┐
                │ AgentResult │
                │ - output    │
                │ - steps     │
                │ - tokens    │
                │ - status    │
                └────────────┘
```
